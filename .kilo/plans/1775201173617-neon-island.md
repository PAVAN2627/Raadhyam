## Implementation Plan

### Step 1: Update User Schema
Modify `Backend/models/users.js` to add:
- `plan`: String, default: "Free" (Free | Monthly Premium | Annual Premium)
- `status`: String, default: "Active" (Active | Inactive | Suspended | Deleted)
- `streams`: Number, default: 0

### Step 2: Implement Dashboard Stats Endpoint
Create controller function for `GET /api/admin/dashboard/stats`:
- Query: Course.count(), User.count(), Enrollment.count(), MusicNote.count()
- Calculate published courses count
- Get recent enrollments (populate user + course)
- Get popular courses (by enrolledStudents)
- Calculate monthlyRevenue from enrollment purchasedPrice (sum of all enrollments in current month)

### Step 3: Implement User Management Endpoints
Add to `Backend/routes/AdminRoutes.js`:
- `GET /api/admin/users` - with pagination, search, role/status/plan filters
- `POST /api/admin/users` - create with hashed password
- `GET /api/admin/users/:id`
- `PUT /api/admin/users/:id`
- `DELETE /api/admin/users/:id`
- `PUT /api/admin/users/:id/status`

Add controller functions in `Backend/controllers/AdminController.js`.

### Step 4: Update Course List with Pagination
Modify existing `getAllCoursesAdmin` to support:
- `page`, `limit` query params
- `search`, `category`, `status` filters

### Step 5: User Deletion (Soft Delete)
- DELETE endpoint sets user status to "Deleted" instead of removing the record
